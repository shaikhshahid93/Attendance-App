# 📱 Smart Attendance App

> An efficient Android application designed to streamline the attendance marking process for students and teachers.

## 📖 About The Project
This project is an **Attendance Management System** developed as a final year project using **Android Studio**. It replaces traditional pen-and-paper attendance registers with a digital mobile solution. The app allows faculty to easily mark attendance, view student lists, and store records safely in a local database.

**Key use cases:**
* Daily attendance marking for lectures/labs.
* Viewing attendance history.
* Managing student profiles (Add/Edit/Delete).

## ✨ Key Features
* **User-Friendly Interface:** Clean and intuitive UI designed with XML.
* **Mark Attendance:** Simple checkbox/toggle system to mark students present or absent.
* **Student Management:** Add, update, or delete student details (Name, Roll No, Class).
* **Date-wise Views:** View attendance records for specific dates.
* **Offline Storage:** Secure local data storage using **SQLite**, ensuring the app works without an internet connection.

## 🛠️ Tech Stack
* **IDE:** Android Studio
* **Language:** Java
* **Frontend:** XML
* **Database:** SQLite
* **JDK:** Java Development Kit 11+

## 🚀 Getting Started

### Prerequisites
* Android Studio (Latest version recommended)
* Android SDK

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/shaikhshahid93/Attendance-App.git](https://github.com/shaikhshahid93/Attendance-App.git)
    ```
2.  **Open in Android Studio:**
    * Open Android Studio -> File -> Open -> Select the cloned project folder.
3.  **Sync Gradle:**
    * Allow Gradle to build and download dependencies.
4.  **Run the App:**
    * Connect your Android device via USB or use an Emulator.
    * Click the 'Run' button (Green Play Icon).
